# Balakumaran D - Cybersecurity Analyst Portfolio

Welcome to my personal portfolio! I am **Balakumaran D**, a Cybersecurity Analyst with a strong background in **SOC (Security Operations Center)**, **Threat Intelligence**, **Penetration Testing**, and **Incident Response**. This repository hosts my online portfolio, showcasing my experience, skills, certifications, and achievements in the field of cybersecurity.

## Overview

This portfolio site is built using **Jekyll** and is hosted on **GitHub Pages**. The theme used is **jekyll-theme-hacker**, which fits my cybersecurity-focused career. The site is designed to highlight my skills, past projects, blog posts, and provide resources such as my **resume** and **certifications** for download.

## Features

- **Cybersecurity Focused**: Highlights my expertise in **Penetration Testing**, **SOC**, **Threat Intelligence**, and **Incident Response**.
- **Portfolio Sections**: Showcase areas like **Security Audits**, **Penetration Testing**, and **Incident Response** with links to detailed case studies and experiences.
- **Social Media Links**: Direct links to my professional social profiles (LinkedIn, GitHub, Twitter, etc.) for easy connection.
- **Downloadable Resources**: Provides an option for visitors to download my **resume** and view my **certifications**.
- **Responsive Design**: Optimized for both desktop and mobile viewing, providing an excellent user experience across devices.
- **Google Analytics**: Integrated for tracking site traffic and performance (if enabled).
